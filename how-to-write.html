How to write HTML?

<!DOCTYPE html> ကြေငြာချက်သည် ဤစာရွက်စာတမ်းသည် HTML5 စာရွက်စာတမ်းဖြစ်ကြောင်း သတ်မှတ်သည်။

<html> သည် HTML file ရဲ့ root element ဟုဆိုလိုသည်။

HTML စာရွက်စာတမ်းတစ်ခုတွင် အဓိကအပိုင်းနှစ်ခုပါဝင်သည်- <head> နှင့် <body> ။ တစ်ခုချင်းစီတွင်၎င်း၏ကိုယ်ပိုင်ခွင့်ပြုထားသောအစိတ်အပိုင်းများနှင့်လိုအပ်ချက်များရှိသည်။

<body>အပိုင်း တွင် နှစ်ပိုင်းရှိသည်- The Text Levelနှင့် The Block Level ။

HTML စာရွက်စာတမ်းတိုင်းအတွက် ဤ tagများ ကို လိုအပ်ပြီး HTML document တစ်ခုလုံးတွင် တစ်ကြိမ်သာ အသုံးပြုသင့်သည်။

File တစ်ခုတည်ဆောက်ခြင်းကို ဤvideo တွင် လေ့လာနိုင်သည်။

Head Section

HTML file တစ်ခု၏ <head> သည် စာမျက်နှာကိုတင်သည့်အခါ Web browser တွင် ပြသခြင်းမရှိသည့်အပိုင်းဖြစ်သည်။ ၎င်းတွင် စာမျက်နှာ <title> ၊ CSS Linkများ (သင်၏ HTML အကြောင်းအရာကို CSS ဖြင့် ပုံစံဆင်ရန် ရွေးချယ်ပါက)၊ စိတ်ကြိုက် favicons Linkများနှင့် အခြားသော Meta data များ (စာရေးသူကဲ့သို့ HTML နှင့်ပတ်သက်သော ဒေတာနှင့် အရေးကြီးသော့ချက်စာလုံးများကဲ့သို့သော အချက်အလက်များပါရှိသည့် စာတမ်းကို ဖော်ပြသည်။) Web Browser များသည် HTML စာရွက်စာတမ်းကို မှန်ကန်စွာပြန်ဆိုရန် <head>တွင်ပါရှိသော အချက်အလက်ကို အသုံးပြုသည်။

HTML <head> element သည် အောက်ပါ elementများအတွက် Container တစ်ခုဖြစ်သည် - <title>, <style>, <meta>, <link>, <script>, နှင့် <base>.

Body Section

HTML <body> tag သည် browser တွင်ပြသသည့် HTML document တစ်ခု၏ အဓိကအကြောင်းအရာကို သတ်မှတ်သည်။ ၎င်းတွင် စာသားအကြောင်းအရာ၊ စာပိုဒ်များ၊ ခေါင်းစီးများ၊ ရုပ်ပုံများ၊ ဇယားများ၊ လင့်ခ်များ၊ ဗီဒီယိုများ စသည်တို့ ပါဝင်နိုင်သည်။

သည် tag ပြီးနောက် ဒုတိယ element ဖြစ်ရမည် သို့မဟုတ် ၎င်းကို </head> နှင့် </html> tag များကြားတွင် ထားရှိသင့်သည်။

Block Level ELement

Headings

  • h1 - Level 1 header
  • h2 - Level 2 header
  • h3 - Level 3 header
  • h4 - Level 4 header
  • h5 - Level 5 header
  • h6 - Level 6 header

Lists

  • ul - Unordered list
  • ol - Ordered list
  • li - List item
  • dl - Definition list
  • dt - Definition term
  • Ddd- Definition

Text containers

  • p - Paragraph
  • pre - Preformatted text
  • blockquote - Large quotation
  • address - Address information

Others

  • div - Logical division
  • center - Centered division
  • form - Input form
  • hr - Horizontal rule
  • table - Tables

Text Level ELement

  • a - Marks content that acts as a hyperlink
  • strong - Formats text to appear in bold
  • code - Marks content that represents computer code
  • br - Represents a line break within grouping elements
  • em - Formats text to appear in italics
  • u - Underlines text
  • span - Used to add hooks to inline text
  • q - Adds quotation marks to text
  • sub - Marks text that should be treated as subscript
